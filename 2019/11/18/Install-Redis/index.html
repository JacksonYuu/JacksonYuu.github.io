Install Redis
（安装Redis）

十一月 18, 2019
  • Install
  • Redis

Redis can be installed in windows and Linux. It can also build a redis cluster when there is a lot of data to be stored. This is just about installing in Linux.

（Redis可在windows和linux中安装，同样在需要存储的数据很多时，也能够搭建一个Redis集群。这里仅仅只介绍在linux中安装。）

Download all files（所有文件下载）：https://pan.baidu.com/s/1oTJchlpCcm0XDh72eM8hTA

Extraction Code（提取码）：xtfw

1、first（第一步）

Upload the tar package downloaded from windows to the virtual machine (using CentOS 7.0 here) and then use （将windows下载的tar包上传到虚拟机（这里使用的是CentOS7.0）当中，之后在使用）

2、second（第二步）

Because redis is written based on C language, it is necessary to install C language in Linux system.

(由于Redis是基于c语言编写的，所以在linux的系统中需要安装c语言。)

3、third（第三步）

Directly enter the redis folder and use the command to complete the installation.

(直接进入Redis的文件夹中，使用命令即可完成安装。)

4、fourth（第四步）

Test whether the installation is successful.（测试是否安装成功）

Create the redis running directory, and copy the scripts for starting services, terminal operations, and configuration files to the directory.

创建Redis的运行目录，并将启动服务的脚本、终端操作的脚本以及配置文件复制到该目录。
