What is Redis？（什么是Redis？）

Redis (move dictionary server) is a key value storage system, similar to memcached. To ensure efficiency, data is cached in memory. But redis supports more types of data storage, including string (string), list (linked list), set (set), Zset (sorted set) and hash (hash type). Moreover, redis periodically writes the updated data to disk or writes the modification operation to the additional record file, and implements master slave synchronization on this basis.

（Redis（REmote DIctionary Server）是一个键值对（ key-value ）形式的存储系统，和 Memcached类似，为了保证效率，数据都是缓存在内存中。但是redis支持更多类型的数据存储，包括string(字符串)、list(链表)、set(集合)、zset(sorted set –有序集合)和hash（哈希类型），并且redis会周期性的把更新的数据写入磁盘或者把修改操作写入追加的记录文件，并且在此基础上实现了master-slave(主从)同步。）

Advantages（优点）

  • High performance（性能高）

    redis can support more than 100k + reading and writing frequency per second.

    （Redis 能支持超过 100K+ 每秒的读写频率。）

  • Rich data types（数据类型多）

    redis supports string, lists, hashes, sets and ordered sets data type operations of binary cases.

    （Redis 支持二进制案例的 Strings，Lists，Hashes，Sets 及 Ordered Sets 数据类型操作。）

  • Atom（原子性）

    all operations of redis are atomic. At the same time, redis also supports the atomic execution of several operations after full merging.

    （Redis 的所有操作都是原子性的，同时 Redis 还支持对几个操作全并后的原子性执行。）

  • Rich features（特性多）

    redis also supports publish / subscribe, notification, key expiration and other features.

    （Redis 还支持 publish/subscribe，通知，key 过期等等特性。）
