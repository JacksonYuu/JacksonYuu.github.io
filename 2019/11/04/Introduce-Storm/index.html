What is Storm？（什么是Storm？）

Storm is a distributed real-time big data processing framework, known as the real-time version of Hadoop in the industry. As more and more scenarios cannot tolerate high latency of Hadoop MapReduce, such as website statistics, recommendation system, early warning system, financial system (high-frequency trading, stock), etc. Therefore, the application of real-time processing (stream computing) for big data is becoming more and more extensive, and storm is occupying the absolute mainstream in real-time processing of big data.

（Storm是分布式实时大数据处理框架，被业界称为实时版Hadoop。随着越来越多的场景对Hadoop的MapReduce高延迟无法容忍，比如网站统计、推荐系统、预警系统、金融系统(高频交易、股票)等等。所以对于大数据实时处理（流计算）的应用日趋广泛，而Storm更是在实时处理大数据中占领着绝对的主流。）

Storm components（Storm的组件）

  • Topologies：Computing topology, also known as a storm task.

    （计算拓扑，也称之为一个storm任务）

  • Spouts：Message source for topology.

    （拓扑的消息源）

  • Bolts：Logical unit of topology processing.

    （拓扑的处理逻辑单位）

  • Tuples：Message tuple. Message tuple, the format used to send messages.

    （消息元组。发送消息所使用的格式）

  • Streams：Data stream.

    （数据流）

  • Streams groupings：Grouping strategy of data flow.

    （数据流的分组策略）

  • Tasks：Task processing unit. Each spin and bolt is executed by several tasks.

    （任务处理单位。每个 Spout 和 Bolt 都由若干个task来执行。）

  • Executor：An executor can have one or more tasks.

    （工作线程。一个executor可以有一个或多个task。）

  • Workers：Worker process.A worker has one or more tasks.

    （工作进程。一个worker有着一个或多个task）

  • Configuration：Configuration of topology.

    （拓扑的配置）
