HBase has three operating modes: stand-alone mode, pseudo-distributed mode and fully distributed mode. In stand-alone mode, you don’t need to build a Hadoop environment. We introduce the installation of Hadoop in fully distributed mode, so we need to build a fully distributed mode Hadoop environment first.HBase has its own Zookeeper service, or can install Zookeeper itself .
