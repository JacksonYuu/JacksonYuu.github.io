Configuration Details（配置详解）

name（配置项名称） effect（作用） example（举例）
agent flume节点名称
agent.sources（监听源） 收集数据并发送给channel agent.sources=r1 r2
agent.channels（临时通道） 临时存放收集的数据 agent.channels=c1 c2
agent.sinks（接收器） 从channel中读取数据并发送给目标 agent.sink=s1 s2
agent.sources.r1.type 设置source名称的源类型 agent.sources.r1.type=avro agent.sources.r2.type=spooldir
agent.sources.r1.channels 设置临时存放的channel agent.sources.r1.channels=c1 agent.sources.r2.channels=c2
agent.channels.c1.type 设置channel名称的类型 agent.channels.c1.type=memory
agent.sinks.s1.type 设置sink名称的类型 agent.sinks.s1.type=logger
configuration（参数） effect（作用） example（举例）
–conf 或 -c 指定配置文件的文件夹名称 -c conf
–conf-file 或 -f 指定配置文件的地址 -f conf/flume-conf.properties
–name 或 -n 指定启动的flume节点名称 -n agent
